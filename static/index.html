Ukázka - Jen Windy

Obsah veřejně dostupného iframu embed.windy.com/embed2.html je prohnán přes proxy, aby obsah iframu měl stejný origin. Následně jsme schopni přistoupit přes iframe.contentWindow.W k API inicializovanému uvnitř iframu.

Díky tomu si přepneme na v iframu běžně dostupnou radarovou vrstvu a odchytáváme událost klik do mapy. V této ukázce také měníme implementaci W.labelsLayer přidáním funkcionality dotahující informace o teplotách ve městech, kterou máte na windy.com. Informace o teplotách ženeme přes proxy, aby nebylo vidět, že voláme vaše API. Zvažujeme tyto teploty na svém backendu na nějakou dobu kešovat, abychom odstínili vaše API od přímé návštěvnosti našich uživatelů.
